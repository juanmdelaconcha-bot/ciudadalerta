📧 Configurar notificaciones
Pasos para activar: 1. Ve a emailjs.com → crea cuenta gratis con Gmail
2. Email Services → Add Service → Gmail → copia el Service ID
3. Email Templates → Create Template → usa las variables de abajo → copia el Template ID
4. Account → copia tu Public Key
5. Pega los tres valores aquí abajo y guarda
Variables para tu template:
{{tipo}} — categoría
{{titulo}} — título de la denuncia
{{descripcion}} — descripción
{{direccion}} — dirección
{{fecha}} — fecha
{{coordenadas}} — lat, lng
{{to_email}} — tu correo
